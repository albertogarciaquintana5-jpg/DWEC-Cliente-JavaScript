Métodos de Arrays — Alberto García Quintana

1. Array.from()

Convierte iterables en arrays → puede aplicar función callback.

const string = "12345"; const array = Array.from(string, num => Number(num) * 2); console.log(array); // [2, 4, 6, 8, 10]

2. Array.of()

Crea un array a partir de argumentos.

const array = Array.of(1,2,3,4,5); console.log(array);

3. Array.at()

Devuelve el elemento del índice (acepta índices negativos).

const array=[10,20,30,40,50]; console.log(array.at(2)); // 30 console.log(array.at(-1)); // 50

4. Array.prototype.with()

Crea una copia reemplazando un índice.

const array=[10,20,30,40,50]; const newArray=array.with(2,99); console.log(newArray);

5. Array.unshift()

Añade al inicio del array.

6. Array.shift()

Elimina el primer elemento.

7. Array.push()

Añade al final del array.

8. Array.pop()

Elimina el último.

9. Array.fill()

Rellena posiciones con un valor.

10. Array.concat()

Combina arrays.
